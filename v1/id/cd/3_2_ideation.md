##3.3.2 Ideation 概念推敲

与前面的大项目不同，概念设计小项目不需要类似UCD项目中的CT+CT / CS+CS 这般完整的设计流程，也可以没有关于 User Experience 的描述内容。这里要做的仅仅是把项目上唯一的亮点表达清楚即可。
